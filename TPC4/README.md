# TPC4

Este trabalho de casa consistiu no desenvolvimento de uma página web, com recurso a HTML, de tema à escolha. Optei por criar um, reading tracker, assim, a página funciona como um dashboard literário onde o utilizador pode visualizar o seu progresso de leitura e as suas coleções.

As principais partes do trabalho são:

- Painel de Estatísticas: Uma visão geral do desempenho de leitura, incluindo percentagens de livros lidos e o rating médio das leituras.

- Controlo de Metas: Uma barra de progresso visual que indica quantos livros faltam para atingir a meta mensal.

- Organização por Categorias: O conteúdo está dividido em secções temáticas (Suspense, Fantasia, Romance e Wishlist), facilitando a navegação.

- Cards de Livro: Cada livro possui uma área dedicada com a capa, autor, avaliação por estrelas e uma etiqueta de estado (Lido, A ler, Quero ler).

## Implementação em HTML
A página contém:

- Navegação Lateral: Sidebar fixa com âncoras internas para permitir que o utilizador salte rapidamente para o género que deseja consultar.

- Secções de Conteúdo: Uso de tags `<section>` e `<div>` para separar claramente cada categoria literária.
