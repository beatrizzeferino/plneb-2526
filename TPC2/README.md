# TPC2

Este trabalho de casa consistiu na utilização expressões regulares utilizando o módulo re.

Foram exploradas diferentes funções, nomeadamente:

* re.match() – verificação de correspondência no início da string;
* re.search() – procura de correspondência em qualquer posição;
* re.findall() – obtenção de todas as ocorrências de um padrão;
* re.sub() – substituição de padrões encontrados;
* re.split() – divisão de strings com base num padrão.

É de ressaltar que:
- À exceção dos casos em que foi explicitamente descrito que deveria ser case insensitive, os exercícios foram realizados como case sensitive. Por exemplo, no exercício 1.4 apenas foram substituidas as ocorrências de 'hello', não incluíndo variações como 'Hello' ou 'HELLO'.
